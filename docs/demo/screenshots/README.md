# Demo Screenshots

These screenshots were captured from a clean local demo run using the Iris UCI data under `datasets/IrisUCI/`.

The source `iris.data` file is headerless, so the screenshot run used temporary CSV copies with these columns:

```text
sepal_length_cm,sepal_width_cm,petal_length_cm,petal_width_cm,class
```
